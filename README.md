# aigw-archive

> 백석대학교 서건우 교수 강의·연구·아티팩트 영구 아카이브
> 🔗 사이트: https://aigw2026-cpu.github.io/aigw-archive/

---

## 📋 이 아카이브는

- 매 학기 강의자료를 주차별로 정리하여 영구 보관합니다
- 학생 작품은 동의 후 후배 학습 사례로 공개됩니다
- 졸업 후에도 언제든 삭제·변경 요청 가능합니다

---
