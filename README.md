## 📋 이 아카이브는

- 매 학기 강의자료를 주차별로 정리하여 영구 보관합니다
- 학생 작품은 동의 후 후배 학습 사례로 공개됩니다
- 졸업 후에도 언제든 삭제·변경 요청 가능합니다

## 📤 학생 작품 제출 방법 (로그인 없이 가능)

> Google 계정(Gmail)으로 로그인 후 아래 폼에서 파일을 직접 제출할 수 있습니다.
> GitHub 계정은 필요하지 않습니다.

### 1단계 — 동의 폼 작성 및 파일 제출

| 과목 | 제출 링크 |
|------|----------|
| 🏃 장애별체육교육론 (2026-1학기) | https://docs.google.com/forms/d/15pF96XuPOe2WvUphCcnsrTRbxIpuq5od4GcobuXP2Vg/viewform |
| 🤸 장애인스포츠론 (2025-1학기) | https://docs.google.com/forms/d/e/1FAIpQLSdbQjnctvkSbc1X-lE-qfE2EwUGeiA6-M74gmLLM66-0HZVHA/viewform |

### 2단계 — 폼에서 직접 파일 첨부

1. 위 링크 클릭
2. 이름, 학번 입력
3. 작품 표기 방식 선택 (실명 / 이니셜 / 익명)
4. 인터넷 공개 동의 여부 선택
5. 동의 확인 체크
6. **작품 파일 첨부** (HTML, PDF, 이미지 등, 최대 10MB)
7. 제출 클릭 → Google Drive에 자동 저장 완료

### 3단계 — 또는 Google Drive에 직접 업로드

> 폼 제출이 어렵거나 파일만 먼저 올리고 싶을 경우 Drive에 직접 업로드할 수 있습니다.

📁 **학생 작품 Drive 폴더 (직접 업로드)**
https://drive.google.com/drive/folders/1m223px5YDwmVg5ISPkmCqD4OhwTWP8YEFwbK08jY1dx9yt5K0gDBhooJY8hF-TwX-ls-tu3U?usp=sharing

업로드 방법:
1. 위 링크 클릭
2. Google 계정으로 로그인
3. 파일을 드래그앤드롭 또는 [+ 새로 만들기] → [파일 업로드] 클릭
4. 업로드 완료 후 동의 폼도 별도 제출 권장

### 4단계 — 제출 확인

폼 제출 시 Google에서 자동으로 확인 이메일이 발송됩니다.
